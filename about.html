About This Project

Student Course Planner is a website that helps students organize their courses, assignments, deadlines, and academic goals.

Our Goal

Our goal is to create a simple, useful, and user-friendly planner that helps students stay organized and manage academic work more efficiently.

Main Features

  • Task creation, editing, deletion, and filtering
  • Persistent task data using localStorage
  • Dynamic course and calendar pages using JavaScript
  • Responsive layout using Bootstrap and a global stylesheet
